PixelRunner is a retro gaming setup that transforms a Raspberry Pi into an arcade console. It uses MAME (Multiple Arcade Machine Emulator) and a custom front-end for intuitive navigation and gameplay using an arcade stick. Designed for quick boot-up, offline use, and ease of play, PixelRunner delivers a nostalgic gaming experience powered by modern hardware.

  • Custom front-end launches on Raspberry Pi boot
  • Arcade stick controls game navigation and launching
  • ROM files automatically detected and categorized
  • Supports metadata display and genre filtering
  • Designed for offline use and ease of setup
  • Raspberry Pi 4 (or other version)
  • Arcade Stick (or other gaming controller)
  • USB-C power supply
  • Micro HDMI cable
  • MicroSD card
  • USB data stick
