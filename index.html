Project Overview

Pixel Runner is a custom-built arcade interface system that runs on a Raspberry Pi. The system features a plug-and-play experience using an arcade stick to browse, launch, and play classic MAME games through a clean, responsive front-end. Designed to boot quickly and operate offline, the system supports USB-based ROM loading and simple user configuration.

Team Members

Our project was developed by a collaborative team of four systems engineering students: Chris Cleare, Pat Harris, Liz Ryan, and Desirae Sanderson. Each member contributed to both the technical implementation and user-focused design of Pixel Runner.

Key Features

  • Fast boot-up and direct launch of arcade front-end
  • USB stick ROM importing
  • Intuitive navigation with arcade stick
  • Game metadata and categorization
  • Offline support, no internet required
  • Robust error handling for missing/corrupt ROMs

Required Components

  • Raspberry Pi 4 (or other supported version)
  • Arcade Stick or compatible USB controller
  • USB-C power supply
  • Micro HDMI cable
  • MicroSD card (16GB or larger)
  • USB data stick for ROM transfers
